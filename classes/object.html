# string

TIP

返回值类型：String

  • 将对象转为String
${objVar.string()}

# jsonpath

参数名 描述 可否为空
path jsonpath 否

TIP

返回值类型：Object

  • 根据JSONPath提取数据
${objVar.jsonpath('$.code')}

# sleep

参数名 描述 可否为空
time 睡眠时间(毫秒) 否

TIP

返回值类型：Object，即返回调用者对象

  • 睡眠500ms
${objVar.sleep(500)}
